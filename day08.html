Day08. 마케팅(2) -글로벌 마케팅-
marginal : 약간의,주변의
means : 방법,수단
by means of : ~에 의하여,~으로
resolve : (문제등을)해결하다
forward : 앞으로,(물건,정보를)보내다
mastermind : (계획등의)입안자,지도자
belong to : ~에 속하다, ~의 소유물이다
put up with : ~을 참다,견디다
ample : 충분한,풍부한
despair : 절망
driven : 의욕이 넘치는
feasible : 실행 가능한
rave review : 호평,극찬하는 기사
come across : 우연히 만나다
mobility : 이동성
fabulous : 굉장히 좋은
stark : (차이가)극명한,황량한
steadiness : 착실함,끈기
alluring : 유혹하는
dissipate : 낭비하다,흩뜨리다
elicit : 이끌어내다
